# Match-3 Game / 消消乐游戏

[English](#english) | [中文](#chinese)

<a name="english"></a>

## English

A classic Match-3 puzzle game built with Vite, TypeScript, and HTML5 Canvas.

### Features

- **30 Levels**: Difficulty increases with each level.
- **Visuals**: Neon-style graphics with glow effects and dynamic fruit themes per level.
- **Mechanics**: Swap tiles to match 3 or more of the same type.
- **Progression**: Target scores and limited moves per level.
- **Audio**: Background music and sound effects (generated via Web Audio API).

### How to Play

1.  Click/Touch a tile to select it.
2.  Click/Touch an adjacent tile to swap.
3.  Match 3 or more tiles of the same type to clear them.
4.  Reach the target score before running out of moves to advance to the next level.

### Development

1.  Install dependencies: `npm install`
2.  Run development server: `npm run dev`
3.  Build for production: `npm run build`

## 中文 (Chinese)

一款使用 Vite, TypeScript 和 HTML5 Canvas 构建的经典消消乐游戏。

### 特性

- **30 个关卡**：难度随关卡增加。
- **视觉效果**：霓虹风格图形，带有发光效果，每关都有不同的水果主题。
- **游戏机制**：交换方块以匹配 3 个或更多相同类型的方块。
- **进程**：每关都有目标分数和步数限制。
- **音频**：背景音乐和音效（通过 Web Audio API 生成）。

### 玩法说明

1.  点击/触摸一个方块进行选择。
2.  点击/触摸相邻的方块进行交换。
3.  匹配 3 个或更多相同类型的方块以消除它们。
4.  在步数用完之前达到目标分数以进入下一关。

### 开发

1.  安装依赖：`npm install`
2.  运行开发服务器：`npm run dev`
3.  构建生产版本：`npm run build`
